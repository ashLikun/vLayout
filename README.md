CustomMvp项目简介 项目Mvp框架 基类activity和fragment

## 使用方法

build.gradle文件中添加:

```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```

并且:

```gradle
dependencies {
    implementation 'com.github.ashLikun:VLayout:{latest version}'
}
```
